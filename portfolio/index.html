Objectives

The goal of this web document is two-fold: a) to provide a summary of who I am and where I'm at professionaly and b) as a guide of what and how I am progressing toward more marketable skills as a computer professional going forward. My goal is to be geared more toward computer web application skills then my previous experiences were. So I'm focusing my training and learning in that area of computer skills. I plan to continue my computer application learning progress and then perhaps to eventually work for others on a part-time contracting or full-time basis using newly acquired skills.

The image above and what I'm using as an icon on each web page is based on the fact that phi times pi squared is almost equal to 2 to the 4th power (Φ * Π * Π ≅ 24). I was fascinated by that unusual fact and a very kind person (Karl Roe) made an image, with the help of Adobe Illustrator, as an illustration of that fact.

This Web document is arrainged into 7 main pages; Home, Skills, Learning, Contact, About me, Resume, and Fun. You are currently reading the "Home" page, which serves as an introduction describing the basic goal (why I made this document) and the general layout (how it is constructed). Another goal of this Web document is to provide contact information for connecting with me or providing feedback which you feel may be helpful for improvements to the document. Obviously, there is contact information at the bottom of each main page of the document, but also there is a "Contact" page for providing feedback directly rather than utilizing Github, Linked-In or E-main mechanisms.
